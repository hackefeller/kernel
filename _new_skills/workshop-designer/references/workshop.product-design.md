# Product Design Workshop

**Duration:** 1-2 days\
**Best for:** Teams ready to turn ideas into tangible concepts\
**Investment:** $10,000 - $25,000

---

## Overview

Transform fuzzy ideas into concrete designs that you can test with real users.

Have a feature idea but not sure how it should work? Redesigning a broken experience? This workshop takes your team from concept to clickable prototype in 1-2 days.

We'll use proven design thinking techniques to explore multiple solutions, test assumptions, and create prototypes you can put in front of users immediately.

---

## What We Do

### Problem Definition

Get crystal clear on what we're solving. Half the battle is articulating the problem correctly.

### Solution Exploration

Generate dozens of ideas rapidly using techniques like Crazy 8's and How Might We exercises.

### Concept Selection

Identify the most promising directions through structured critique and dot voting.

### Rapid Prototyping

Create clickable prototypes in hours, not weeks. No code required—it just needs to feel real.

### Validation Planning

Design tests to prove (or disprove) your assumptions with real users.

---

## Facilitation Techniques

**How Might We's**\
Reframe problems as opportunities to unlock creative solutions.

**Crazy 8's**\
8 different ideas in 8 minutes. Quantity breeds quality when you force rapid ideation.

**Dot Voting**\
Democratic decision-making that moves fast. Everyone gets votes to allocate.

**Storyboarding**\
Map the complete user experience from beginning to end.

**Paper Prototyping**\
Test ideas before writing code using sketches and low-fidelity mockups.

## Workshop Agenda

### Day One

**Morning Session (3-4 hours)**

**9:00 - 9:30:** Kickoff & Context

- Problem overview
- User research review
- Success criteria

**9:30 - 10:30:** Problem Framing

- How Might We exercises
- Problem statement refinement
- Success metrics definition

**10:30 - 10:45:** Break

**10:45 - 12:30:** Ideation

- Lightning demos (review existing solutions)
- Individual sketching (Crazy 8's)
- Group sharing and clustering

**12:30 - 1:30:** Lunch

**Afternoon Session (3-4 hours)**

**1:30 - 2:30:** Concept Development

- Refine strongest ideas
- Create storyboards
- Map user flows

**2:30 - 3:30:** Critique & Selection

- Present all concepts
- Structured critique
- Dot voting to select direction

**3:30 - 3:45:** Break

**3:45 - 5:00:** Prototyping Kickoff

- Break into workstreams
- Start building prototype
- Assign responsibilities

### Day Two (if 2-day workshop)

**Morning Session (3-4 hours)**

**9:00 - 12:00:** Prototype Building

- Continue building clickable prototype
- Test internally
- Iterate based on feedback

**12:00 - 1:00:** Lunch

**Afternoon Session (3-4 hours)**

**1:00 - 2:30:** Refinement

- Polish prototype
- Add realistic content
- Test all interactions

**2:30 - 3:00:** Break

**3:00 - 4:30:** Testing Plan

- Define research questions
- Create testing script
- Identify participants

**4:30 - 5:00:** Wrap-up

- Review deliverables
- Next steps
- Documentation review
