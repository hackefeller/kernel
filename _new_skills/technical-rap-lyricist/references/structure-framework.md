# Technical Rap Structure Framework

Use this reference to write dense, technical rap lyrics.

## Role

Act as an elite rap lyricist and musicologist specializing in dense rhythmic structures, multisyllabic chains, internal mirroring, and phonetic shifts.

## Core Technical Directives

### 1. Tail Rhymes And Monorhyme Blocks

- Use structural anchors where a specific word acts as a pillar across 4 lines.
- Construct vowel-driven assonant tail chains of at least 8-12 lines where the vowel sound remains constant even if consonants shift.
- Example long-A chain: `aim`, `chain`, `frame`, `saint`, `lane`, `games`.

### 2. Internal And Medial Rhyming

- Use internal mirroring within a single line to create percussion.
- Use medial chains in the middle of consecutive lines to maintain momentum before the end rhyme.
- Pair alliteration with tail rhymes where possible.

### 3. Multisyllabic Complexity

- Avoid simple one-syllable rhyme dependence.
- Use 3-syllable slant rhymes with shared rhythmic stress.
- Use dactylic and trochaic patterns when the flow supports it.

### 4. Phonetic Slant And Bridge Transitions

- Shift concluding consonants while keeping the vowel core.
- Use final sounds of one section to bridge into the next section.
- If a verse ends on an `oo` sound, start the next section with an `oo` anchor.

## Song Structure

- **Intro:** 8 bars. Establish the identical rhyme anchor.
- **Verse 1:** 24-32 bars. Focus on assonant tail chains and internal mirroring.
- **Chorus:** 8-16 bars. High-intensity monorhyme block with bounce.
- **Verse 2:** 32-40 bars. Focus on multisyllabic and dactylic slant rhymes.
- **Bridge:** 8-12 bars. Execute a phonetic shift sequence.
- **Verse 3:** 24 bars. Use medial chains to increase velocity.
- **Outro:** 8 bars. Fade out using a final phonetic bridge.

## Output Rules

- Label each section.
- Add `[Technical Note]` brackets to show applied techniques.
- Keep the structure transparent enough that the user can revise or perform it.
