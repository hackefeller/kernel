Check existing knowledge before creating new notes.

Use `kernel knowledge list` to see existing notes. Browse `.kernel/knowledge/notes/`, `.kernel/knowledge/guides/`, `.kernel/knowledge/reference/`, and `.kernel/knowledge/learnings/` directly to read content.
